## ✨ Features

- 🤖 **AI-Powered** — Powered by Claude (Anthropic) with full WingmanX brand knowledge
- 🏍️ **WingmanX Branded** — Custom logo, orange/silver color scheme, Bebas Neue typography

## 🌐 Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Click **Save**
5. Your chatbot will be live at `https://<your-username>.github.io/<repo-name>/`
